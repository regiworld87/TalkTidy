# TalkTidy 🗣️✨

A City of Heroes Homecoming chat log parser. Drag in your chat logs and filter, color-code, and export them.

**100% client-side** — your logs never leave your machine.

## Features
- Parse Homecoming chat logs (`(channel) [Timestamp] Speaker: message`)
- Filter by speaker (checkboxes + search)
- Toggle system/emote chatter
- 4 color schemes for channels
- Copy as plain text or export as styled HTML
- Token system (5 free parses)
